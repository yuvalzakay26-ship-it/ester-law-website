כל הטקסטים באתר מרוכזים בקובץ אחד: `src/content/siteContent.js`.
יש לפתוח קובץ זה ולעדכן את השדות הבאים בתוך האובייקט `brand`:

- [ ] **שם העסק:** ודא ש-"אסתר גישור" הוא השם הסופי.
- [ ] **טלפון:** החלף את `050-1234567` במספר האמיתי.
- [ ] **אימייל:** החלף את `office@ester-law.co.il` בכתובת האמיתית.
- [ ] **כתובת:** החלף את `רחוב המגשרים 10, תל אביב` בכתובת הקליניקה (או מחק אם אין קבלת קהל).
- [ ] **WhatsApp:** עדכן את הקישור `https://wa.me/972501234567` (המספר הבינלאומי ללא +).
- [ ] **דומיין:** עדכן את `https://www.ester-mediation.co.il` לדומיין הסופי שנרכש (חשוב ל-SEO).

## 2. התאמת תכנים
עבור על שאר התוכן ב-`siteContent.js` וודא שהוא מדוייק:

- [ ] **שירותים (`services`):** האם רשימת השירותים (גישור משפחתי, עסקי וכו') מקיפה ומדויקת?
- [ ] **תהליך (`process`):** האם שלבי העבודה מתארים נכונה את המציאות?
- [ ] **שאלות נפוצות (`faq`):** מומלץ להוסיף שאלות שלקוחות באמת שואלים.
- [ ] **אודות (`hero`):** ודא שהמשפט המרכזי ("יישוב סכסוכים...") משקף את המסר השיווקי הרצוי.

## 3. SEO ורשתות חברתיות
בתוך `siteContent.js` תחת האובייקט `seo`:

- [ ] **כותרות (Titles):** ודא שהכותרות לכל עמוד מתאימות למילות המפתח שרוצים לקדם.
- [ ] **תיאורים (Descriptions):** ודא שהתיאורים מושכים ומניעים לפעולה (בגוגל).
- [ ] **תמונת שיתוף:** הוחלפה תמונת בסיס ב-`public/og-image.svg`. מומלץ להחליף לקובץ PNG/JPG איכותי יותר עם לוגו אמיתי אם יש.

## 4. בדיקות טכניות סופיות
לפני שמפרסמים את הקישור ללקוחות:

- [ ] **טופס יצירת קשר:** כרגע הטופס הוא **הדגמה בלבד** (Simulated). יש לחבר אותו לשירות שליחת מיילים (כגון EmailJS, Formspree או צד שרת) כדי לקבל את הפניות באמת.
- [ ] **מובייל:** פתח את האתר מהטלפון הנייד וודא שהכל נראה תקין וקריא.
- [ ] **קישורים:** לחץ על כל הקישורים בתפריט ובפוטר וודא שהם מובילים למקום הנכון.
- [ ] **נגישות:** ודא שניתן לנווט באתר בעזרת מקלדת (Tab).

## 5. לאחר העלייה לאוויר
- [ ] חבר את האתר ל-**Google Search Console**.
- [ ] חבר את האתר ל-**Google Analytics**.
- [ ] שלח את הקובץ `sitemap.xml` לגוגל דרך ה-Search Console.
